logos.ai is linked to the original AI files <1024_512.ai, 256.ai, 128.ai, 64.ai, 32_1.ai, 32_2.ai, 16.png> with artboards for each resolution & version.

To save new set of PNG files, Adobe Illustrator > File > Export:
	1. File name "logo"
	2. Save as type "PNG"
	3. Use Artboards "checked"
	For PNG Options:
		1. Resolution "Screen"
		2. Background Color "Transparent"
		3. Anti-aliasing "Art Optimized"
		4. Interlaced "Checked"
